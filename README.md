# Hospital Management Analytics Dashboard

## Project Overview

This project demonstrates an end-to-end Data Analytics workflow using SQL and Power BI. The objective was to analyze hospital operations, patient demographics, revenue performance, doctor efficiency, and feedback ratings to generate actionable business insights.

The project includes data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive Power BI dashboards.

---

## Tools & Technologies

- SQL Server
- Power BI
- Power Query
- DAX
- Excel (Data Source)

---

## Dataset Information

The dataset contains hospital management records including:

- Patient Information
- Doctor Details
- Department Information
- Disease and Treatment Data
- Billing Information
- Payment Modes
- Insurance Providers
- Feedback Ratings
- Admission and Discharge Details

Total Records: 10,000+

---

## Data Cleaning Process

Performed data cleaning using SQL:

- Checked for null values
- Identified duplicate records
- Standardized text fields
- Validated age values
- Verified billing and payment information
- Handled missing insurance provider data
- Validated admission and discharge dates
